# lassification-framework-and-symbolic-representation-of-the-overall-situation.
北京2020高考题21题第三问
我们要证明数列 \{a_n\} 为等比数列。（origin）
等比数列意味着\frac{a_2}{a_1} = \frac{a_3}{a_2} = \frac{a_4}{a_3} = \frac{a_5}{a_4} = \cdots。

已知条件：数列递增，即对于任意正整数 k，有 a_{k+1} > a_k > a_{k-1} > \cdots > a_1。

性质一：对于数列中任意两项 a_i, a_j（其中 i > j），在数列中都存在一项 a_m，使得 \frac{a_i^2}{a_j} = a_m。该性质的意思是 \frac{a_i^2}{a_j} 得到一个确定的值，这个值等于数列中的某一项 a_m。

性质二：对于数列中任意项 a_n（其中 n \geq 3），在数列中都存在两项 a_p, a_q（p > q），使得 a_n = \frac{a_p^2}{a_q}。这里的意思是：对于确定的 a_n（n \geq 3），一定可以找到 a_p, a_q（p > q），使得这个确定的 a_n 等于 \frac{a_p^2}{a_q}。

我们代入 n=3 试一试。在证明 \{a_n\} 为等比数列之前，\{a_n\} 是任意数列，所以我们有理由猜想 a_3 = \frac{a_2^2}{a_1} 也可能等于 \frac{a_5^2}{a_3} 或 \frac{a_{1000}^2}{a_{997}} 等。也就是说，对于 a_3 而言，表达式 \frac{a_p^2}{a_q} 不唯一。

对 a_n = \frac{a_p^2}{a_q} 进行变形，得到 \frac{a_n}{a_p} = \frac{a_p}{a_q}。代入 n=3 得 a_3 = \frac{a_p^2}{a_q}，即 \frac{a_3}{a_p} = \frac{a_p}{a_q}（其中 p > q）。这形同等比数列中 \frac{a_{k+1}}{a_k} = \frac{a_k}{a_{k-1}} = \cdots = \frac{a_2}{a_1} = r。若想证明 \{a_n\} 为等比数列，此时应有 a_p = a_2，a_q = a_1。还会想到等比数列中前后项的其他表达式：a_k = a_{k-1} \times r = a_1 \times r^{k-1}。我们依然会感到混乱。

所以我们可以先假设 \{a_n\} 为等比数列并且满足上述条件，举一个例子：1, 2, 4, 8, 16, 32, 64, 128, \dots。我们发现 a_3 = \frac{a_2^2}{a_1}，a_4 = \frac{a_3^2}{a_2}。

从 a_3, a_4 中我们猜测：如果 \{a_n\} 是一个满足题目条件的等比数列，那么对于 n \geq 3，有 a_n = \frac{a_p^2}{a_q} = \frac{a_{n-1}^2}{a_{n-2}}，即 a_p = a_{n-1}，a_q = a_{n-2}。现在已经举例发现 a_n = \frac{a_{n-1}^2}{a_{n-2}} 对于 n=3,4 成立。如果该式对 n \geq 5 同样成立，我们就能证明 \{a_n\} 为等比数列。

我们可以先通过题目条件证明 a_3, a_4 的表达式。

设 a_3 = \frac{a_p^2}{a_q}，我们的任务是利用题目条件证明 a_p = a_2，a_q = a_1。先固定 a_p。因为 a_p 是实数，我们用数值 a_3 对 a_p 进行划分：a_p > a_3，a_p < a_3，a_p = a_3。然后逐个讨论。

当 a_p = a_3 时，即 p=3。固定 a_p 后讨论 a_q：a_3 = \frac{a_3^2}{a_1} 或 a_3 = \frac{a_3^2}{a_2}。代数变形：a_3 \cdot a_1 = a_3^2，但由递增性 a_1 < a_3，所以 a_3 \cdot a_1 < a_3^2，矛盾；类似地，a_3 \cdot a_2 = a_3^2，但 a_2 < a_3，矛盾。因此排除。

当 p < 3 时，即 a_p < a_3，可能情况为 p=2，此时 a_3 = \frac{a_2^2}{a_1}。由递增性，a_3 > a_2，所以 a_3 \cdot a_1 = a_2^2，且 a_1 < a_2 < a_3，无法直接排除。

当 p > 3 时，固定 a_q = a_1，则 a_3 = \frac{a_4^2}{a_1} 或 a_3 = \frac{a_5^2}{a_1} 等，无法穷尽所有可能。

所以我们被迫只能考虑前两种情况，得到 a_3 = \frac{a_2^2}{a_1}。

讨论 a_4：设 a_4 = \frac{a_p^2}{a_q}，同样分为 p=4，p<4，p>4。当 p=4 时，有 a_4 = \frac{a_4^2}{a_1}，a_4 = \frac{a_4^2}{a_2}，a_4 = \frac{a_4^2}{a_3}，利用递增性可排除。但 a_4 = \frac{a_3^2}{a_2} 和 a_4 = \frac{a_3^2}{a_1} 无法用递增性直接排除。

需要增加限制条件。利用性质一：存在 a_m = \frac{a_i^2}{a_j}。比较 \frac{a_3^2}{a_2} 和 \frac{a_3^2}{a_1} 的大小。不妨推导出 a_4 的范围来看哪个符合。设 a_m = \frac{a_3^2}{a_2}，由于 a_3 > a_2，所以 a_m > a_3，从而 a_m \geq a_4，即 \frac{a_3^2}{a_2} \geq a_4。若 a_4 = \frac{a_3^2}{a_1}，则因为 a_1 < a_2，有 \frac{a_3^2}{a_1} > \frac{a_3^2}{a_2} \geq a_4，矛盾。所以排除 a_4 = \frac{a_3^2}{a_1}，于是得到 a_4 = \frac{a_3^2}{a_2}。

现在我们有了 a_1, a_2, a_3, a_4 成等比数列。我们可以替换目标：证明对 n \geq 3 有 a_n = \frac{a_{n-1}^2}{a_{n-2}} 成立。我们可以直接证明对 k \geq 2 有 a_{k+2} = \frac{a_{k+1}^2}{a_k}，成立即证明了 \{a_n\} 为等比数列。因为变形可得 \frac{a_{k+2}}{a_{k+1}} = \frac{a_{k+1}}{a_k}。如果假设 a_{k+1} 及其前面的项为等比数列，那么 a_{k+2} 将和 a_{k+1}, a_k, a_{k-1}, \dots, a_1 成等比数列。如果 k 不断增加，则成等比的项数也会无限增加。

另外，换个角度：设 a_{k+2} = \frac{a_p^2}{a_q}，即 \frac{a_{k+2}}{a_p} = \frac{a_p}{a_q}。如果要证明 \{a_n\} 为等比数列，则必须有 a_p = a_{k+1}，a_q = a_k。所以我们必须通过排除法，利用题目性质证明 a_p 只能等于 a_{k+1}，a_q 只能等于 a_k。

根据证明 a_3, a_4 时的例子，我们固定 a_p，先讨论变量 a_p。将 a_p 划分为：a_p > a_{k+2}，a_p < a_{k+2}，a_p = a_{k+2}。一旦我们证明 a_p > a_{k+2} 不成立，即意味着对于 a_4, a_5, a_6, \dots，它们的表达式 \frac{a_p^2}{a_q} 中的 a_p 不能取到大于该项的值，即 a_p > a_4，a_p > a_5，a_p > a_6 等都不成立。

先来证明 a_p > a_{k+2} 不成立。假设 a_{k+1} 及其之前的项成等比数列（归纳假设）。设 a_{k+2} = \frac{a_p^2}{a_q}。当 p > k+2，即 p \geq k+3。将情况分成 p > k+3 和 p = k+3。

当 p > k+3 时，固定 a_q = a_1，则 a_{k+2} = \frac{a_{k+4}^2}{a_1} 或 \frac{a_{k+5}^2}{a_1} 等，无法穷尽。

当 p = k+3 时，a_q 无法穷尽。即 p > k+2 的情况无法被穷尽。

当 p = k+2 时，由 a_{k+2} = \frac{a_{k+2}^2}{a_q} 得 a_q = a_{k+2}，与 q < p 矛盾，排除 p = k+2。

当 p < k+2 时，即 p \leq k+1，分成 p = k+1 和 p < k+1。

当 p = k+1 时，固定 a_p，a_{k+2} = \frac{a_{k+1}^2}{a_q}，a_q 讨论不尽，即 p = k+1 的情况讨论不尽。

当 p < k+1，即 p \leq k 时，固定 a_p \leq a_k，则 a_{k+2} = \frac{a_p^2}{a_q} \leq \frac{a_k^2}{a_q}。在这种情况下讨论 a_q 讨论不尽。取一个特殊情况：a_{k+2} \leq \frac{a_k^2}{a_1}，则 a_{k+2} \cdot a_1 \leq a_k^2。但 a_{k+2} \cdot a_1 < a_{k+2}^2，结果表明 a_k^2 可能出现在 a_{k+2}^2 的左侧或右侧，无法利用递增性排除。

增加限制条件，利用性质一。当我们没有思路时，就去找和 a_{k+2} 有关的关系。找 a_m 和 a_{k+2} 的关系。设 a_m = \frac{a_k^2}{a_1}，由于 \frac{a_k^2}{a_1} > a_k，所以 m \geq k+1。

当 m = k+1 时，a_{k+1} = \frac{a_k^2}{a_1}，但此时 a_{k+2} \leq \frac{a_k^2}{a_1} = a_{k+1}，与递增性矛盾，排除。

当 m > k+1 时，m \geq k+2，即 a_m \geq a_{k+2}。分情况：当 m = k+2 时，a_m = a_{k+2} = \frac{a_k^2}{a_1}；当 m > k+2 时，a_m = \frac{a_k^2}{a_1} > a_{k+2}，即 a_{k+2} 可能等于 \frac{a_k^2}{a_{k-1}}，\frac{a_k^2}{a_{k-2}} 等，列举不尽。

先讨论 m = k+2 的情况，即 a_{k+2} = \frac{a_k^2}{a_1}。我们再次没有思路，那就再去找某个和 a_{k+2} 有关的东西。不妨设 a_{m_1} = \frac{a_{k+1}^2}{a_k}，得到 a_{m_1} > a_{k+1}，即 a_{m_1} \geq a_{k+2}。此时想，a_{m_1} - a_{k+2} \geq 0 如何证明？可以代入 a_{k+1} 和 a_k 的表达式。因为已经假设 a_{k+1} 及其之前的项成等比数列，设 a_{k+1} = \frac{a_k^2}{a_{k-1}}，a_k = \frac{a_{k-1}^2}{a_{k-2}}。结果得到 a_{m_1} = \frac{a_k^3}{a_{k-1}^2} = \frac{a_{k-1}^4}{a_{k-2}^3}，而 a_{k+2} = \frac{a_k^2}{a_1}。将两者相减，并利用 a_k = \frac{a_{k-1}^2}{a_{k-2}}，得到 \frac{a_k}{a_{k-1}^2} = \frac{1}{a_{k-2}} < \frac{1}{a_1}（因为 a_1 < a_{k-2}），所以 a_{m_1} - a_{k+2} < 0，从而排除 a_{k+2} = \frac{a_k^2}{a_1}。但只是排除了 m = k+2 的情况，m > k+2 无法穷尽，即 p \leq k 的情况我们无法穷尽。

再看 p = k+1 的情况，讨论 q < k+1，即 q \leq k。固定 p = k+1，则 a_{k+2} = \frac{a_{k+1}^2}{a_q} \geq \frac{a_{k+1}^2}{a_{k-1}}。我们已经知道 a_{m_1} = \frac{a_{k+1}^2}{a_k} = \frac{a_k^3}{a_{k-1}^2}，而 a_{k+2} \geq \frac{a_{k+1}^2}{a_{k-1}} = \frac{a_k^4}{a_{k-1}^3}。将两者相减：当 a_{k+2} = \frac{a_k^4}{a_{k-1}^3} 时，我们发现 \frac{a_k^3}{a_{k-1}^2} 需要乘以 \frac{a_k}{a_{k-1}} 才能等于 a_{k+2}。也就是说，如果公比大于 1（分别讨论a1大于0 a1小于0 a1大于0时且a1到ak加1的项为等比 ak加1大于ak大于...a1时 r大于1

所以我们发现，如果利用 a_{k+2} = \frac{a_p^2}{a_q} 这个式子，结合题目条件去唯一确定 a_p 和 a_q，需要排除所有可能的 a_p 和 a_q 的情况，这很不现实。

我们想到了等比数列的另一个定义：a_{k+1} = a_1 \times r^k，a_k = a_1 \times r^{k-1}。当 p \leq k+1，q \leq k 时，a_p 和 a_q 在 a_{k+1} 及之前项的范围内，可以利用等比数列的性质，所以 a_p = a_1 \times r^{p-1}，a_q = a_1 \times r^{q-1}。代入 a_{k+2} = \frac{a_p^2}{a_q}，得到 a_{k+2} = a_1 \times r^{2p-q-1}。我们必须要确定 a_{k+2} = a_1 \times r^{k+1}。利用递增性，a_{k+2} > a_{k+1} = a_1 \times r^k，即指数 2p-q-1 > k。现在我们只需要再构造出 2p-q-1 \leq k+1 即可。我们利用 a_{m_1} = \frac{a_{k+1}^2}{a_k} = a_1 \times r^{k+1}，且由性质一，存在某个项等于这个值，且其下标至少为 k+2，所以 a_1 \times r^{k+1} \geq a_{k+2} = a_1 \times r^{2p-q-1}，因此 k+1 \geq 2p-q-1。因为 2p-q-1 是正整数，结合 2p-q-1 > k，得到 2p-q-1 = k+1，得证。

,.解题思路

1. 目标与已知条件

我们要证明数列 \{a_n\} 是等比数列，即

\frac{a_2}{a_1} = \frac{a_3}{a_2} = \frac{a_4}{a_3} = \cdots

已知条件：

· 数列严格递增：a_1 < a_2 < a_3 < \cdots
· 性质一：对任意两项 a_i, a_j（其中 i > j），存在某项 a_m 使得
  \frac{a_i^2}{a_j} = a_m
  即 \frac{a_i^2}{a_j} 的值必定等于数列中的某一项。
· 性质二：对任意项 a_n（其中 n \ge 3），存在两项 a_p, a_q（p > q）使得
  a_n = \frac{a_p^2}{a_q}
  即数列中第 n 项（n\ge 3）一定能表示成数列中某两项的平方与商的组合。

2. 从具体项入手试探

先考虑 a_3。由性质二，存在 p>q 使得

a_3 = \frac{a_p^2}{a_q}

变形得

\frac{a_3}{a_p} = \frac{a_p}{a_q}

这形同等比数列中相邻项的比值关系。

2.1 分析 a_3 的可能表达式

我们尝试固定 p，按 p 与 3 的大小关系分类讨论：

· 若 p = 3，则 a_3 = \frac{a_3^2}{a_q}，推出 a_q = a_3，但 q < p = 3，与递增性矛盾。
· 若 p < 3，则 p 只能为 2，此时 q 只能为 1，得到
  a_3 = \frac{a_2^2}{a_1}
  递增性无法直接排除此情况。
· 若 p > 3，则 a_p > a_3，代入等式后 a_q = \frac{a_p^2}{a_3} > a_p，但 q < p，矛盾。

因此唯一可能为 a_3 = \frac{a_2^2}{a_1}。

2.2 分析 a_4 的可能表达式

类似地，设 a_4 = \frac{a_p^2}{a_q}。

· 若 p = 4，得 a_q = a_4，与递增性矛盾。
· 若 p < 4，则 p 可能为 3 或 2。
  · 当 p = 3 时，a_4 = \frac{a_3^2}{a_q}。由性质一（取 i=3, j=2），存在 a_m = \frac{a_3^2}{a_2}，且 a_m > a_3，故 m \ge 4，即 a_m \ge a_4，于是
    \frac{a_3^2}{a_2} \ge a_4
    若 a_4 = \frac{a_3^2}{a_1}，则 \frac{a_3^2}{a_1} > \frac{a_3^2}{a_2} \ge a_4，矛盾。因此唯一可能是
    a_4 = \frac{a_3^2}{a_2}
  · 当 p = 2 时，a_4 = \frac{a_2^2}{a_q}，但 a_4 > a_3 = \frac{a_2^2}{a_1}，故 a_q < a_1，不可能。
· 若 p > 4，类似 a_3 的分析会导出矛盾。

因此得到

a_4 = \frac{a_3^2}{a_2}

3. 发现规律并建立递推猜想

由 a_3 = \frac{a_2^2}{a_1} 和 a_4 = \frac{a_3^2}{a_2} 可得

\frac{a_3}{a_2} = \frac{a_2}{a_1}, \quad \frac{a_4}{a_3} = \frac{a_3}{a_2}

相邻项的比值相等。这提示数列可能为等比数列。

我们猜想：对于任意 n \ge 2，有

a_{n+1} = \frac{a_n^2}{a_{n-1}}

若此猜想成立，则

\frac{a_{n+1}}{a_n} = \frac{a_n}{a_{n-1}}

从而数列是等比数列。

4. 一般情况的归纳证明思路

设数列前 k+1 项已成等比，即存在公比 r = \frac{a_2}{a_1} > 1，使得

a_i = a_1 r^{\,i-1} \quad (i=1,2,\dots,k+1)

现需证明 a_{k+2} = a_1 r^{\,k+1} = \frac{a_{k+1}^2}{a_k}。

由性质二，存在 p>q 使得

a_{k+2} = \frac{a_p^2}{a_q}

4.1 分类讨论的框架

我们按 p 与 k+2 的大小关系分类：

1. p > k+2：利用递增性导出矛盾（整体排除）。
2. p = k+2：快速导出矛盾。
3. p < k+2：进一步细分：
   · 若 p = k+1，则需证明必有 q = k。
   · 若 p \le k，则 p, q 均在归纳假设的范围内，此时可利用等比通项公式。

4.2 关键步骤：利用指数形式统一处理

当 p \le k 时，由归纳假设：

a_p = a_1 r^{\,p-1}, \quad a_q = a_1 r^{\,q-1}

代入得

a_{k+2} = a_1 r^{\,2p-q-1}

同时，由性质一（取 x=k+1, y=k）可得

a_m = \frac{a_{k+1}^2}{a_k} = a_1 r^{\,k+1}

且 a_m \ge a_{k+2}（因为 m \ge k+2）。于是

a_1 r^{\,k+1} \ge a_{k+2} = a_1 r^{\,2p-q-1} \quad \Rightarrow \quad k+1 \ge 2p-q-1

又由递增性：

a_{k+2} > a_{k+1} = a_1 r^{\,k} \quad \Rightarrow \quad 2p-q-1 > k

因此

k < 2p-q-1 \le k+1

由于 2p-q-1 是整数，故必须等于 k+1，从而

a_{k+2} = a_1 r^{\,k+1} = \frac{a_{k+1}^2}{a_k}

4.3 其他情况的处理

· 当 p = k+1 时，可类似证明必须有 q = k，否则推出矛盾。
· 当 p > k+2 或 p = k+2 时，均会导出与递增性矛盾的结论。

5. 结论

由数学归纳法，对所有 n \ge 2 有

a_{n+1} = \frac{a_n^2}{a_{n-1}}

进而

\frac{a_{n+1}}{a_n} = \frac{a_n}{a_{n-1}}

所以数列 \{a_n\} 是等比数列。

6. 方法总结

本题的解题核心在于：

1. 从具体项（a_3, a_4）入手，利用分类讨论和性质排除，得到初始递推关系。
2. 通过观察相邻项比值相等，提出一般递推猜想。
3. 使用数学归纳法证明猜想，其中关键步骤是：
   · 利用性质二将 a_{k+2} 表示为 \frac{a_p^2}{a_q}。
   · 按 p 与 k+2 的大小关系分类，并利用递增性排除不可能情况。
   · 在 p \le k 的情况下，引入归纳假设下的指数形式，将问题转化为整数不等式，从而确定唯一可能
